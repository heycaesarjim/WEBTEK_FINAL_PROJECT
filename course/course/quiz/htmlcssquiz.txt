1. What is the CSS selector which allows you to target every element in a web page? = universal selector (*)
2. Which CSS property allows you to hide an element but still maintain the space it occupies in the web page? = visibility or opacity

4. What is the CSS at-rule that can allow you to define the character encoding of a stylesheet? = @charset
5. Who is making the Web standards? = The World Wide Web Consortium
